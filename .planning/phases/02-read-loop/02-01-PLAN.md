  <behavior>
    generateSnippet(body, maxLen=300):
    - Input: "# My Heading\n\n**Bold** text with `code` here" → Output: "My Heading Bold text with code here" (stripped, no ellipsis if ≤300 chars)
    - Input: body string of 350 stripped chars → Output: truncated at last word boundary before 300, ending with "…"
    - Input: "" → Output: ""
    - Input: body exactly 300 stripped chars → Output: returned as-is, no "…"

    formatEntryDate(createdAt, now):
    - createdAt same calendar day as now → "Today — May 25, 2026"
    - createdAt one calendar day before now → "Yesterday — May 24, 2026"
    - createdAt two days before → "2 days ago — May 23, 2026"
    - Date format: toLocaleDateString('en-US', { month: 'long', day: 'numeric', year: 'numeric' })

    getEntries(): Promise<EntryListItem[]>
    - Returns all entries newest-first (orderBy desc createdAt)
    - Each item has: id, title, snippet (via generateSnippet), dateLabel (via formatEntryDate), tags (string[])
    - Uses LEFT JOIN entries → entryTags → tags; reduces flat rows to per-entry objects using Map (preserves insertion order)
    - tagName null (entry with no tags) → tags array remains []
    - body and createdAt are NOT included in the returned EntryListItem (stripped in final map)

    getEntryById(id): Promise<EntryDetail | null>
    - Returns null when rows.length === 0 (missing entry or NaN id)
    - Returned object includes body (raw Markdown string, not stripped)
    - Same LEFT JOIN + reduce pattern as getEntries()
  </behavior>
